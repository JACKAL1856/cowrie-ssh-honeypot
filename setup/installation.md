**Cowrie Installation**

This document describes the installation of the Cowrie SSH honeypot.

**Environment**
- Host OS: Ubuntu 24.04 LTS (UTM Virtual Machine)
- Honeypot: Cowrie (SSH)
- Attacker Machine: Kali Linux

**Steps**
1. Installed system dependencies
2. Created dedicated `cowrie` user
3. Installed Cowrie using virtual environment
4. Configured SSH honeypot on port 2222
5. Verified Cowrie service is running
